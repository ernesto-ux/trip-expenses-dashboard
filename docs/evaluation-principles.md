# Trip Spending Evaluation Principles

Framework for honestly evaluating whether a trip was well-managed financially. Built from lessons learned on Amsterdam abril 2026.

## Core principle: people-scaled costs

**When evaluating per-decision cost for couples/groups, always multiply by people count.** Single-person heuristics fail.

Example:
- Bolt CDG €40.10 looks "expensive" vs RER B €11.80
- But for 2 people: Bolt = €20.05/person, RER = €23.60 total → Bolt is *cheaper*
- Plus: faster, safer at late hours, with luggage

If you don't apply the people-scaled lens, you'll wrongly criticize rational decisions.

## Evaluation layers

### Layer 1: Anchor costs (typically 60-80% of trip)
- Hotel, flights, anchor event (concert, conference, etc.)
- These are locked-in before arrival
- Evaluate against market rate (€/night, €/flight, €/ticket)
- Not "wasteful" unless dramatically off-market

### Layer 2: Operational efficiency (food, daily transport)
- Food €/person/day in destination context
  - Amsterdam €19/p/day = excellent
  - Paris €25-30/p/day = average
  - SF/NYC €40-50/p/day = average
- Transport: prioritize cost/time/safety **for the group**, not just cheapest
- Day-trip transport (Bolts to specific spots): evaluate ROI on time saved

### Layer 3: Personal/discretionary (shopping, premium upgrades)
- This is where real disparity emerges
- Shopping €X is a *conscious* personal choice, not a "trip overrun"
- Premium upgrades (Gold concert ticket, business class) = personal preference
- These should NOT count as "trip overspend" — they're separate budget decisions

## Disparity analysis

When two people spent very different amounts, decompose:

1. Anchor split (e.g., 450/150 for concert) → conscious choice, no waste
2. Operational (food, hotel) → should be ~equal if split 50/50
3. Personal (shopping, upgrades) → 100% individual responsibility

If disparity is 100% explained by Layers 1+3, the trip itself was neutral. The "extra" spending is not the trip — it's individual.
